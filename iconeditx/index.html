Why IconEditX?

📐 Multi-Frame ICO

Create icons with multiple resolutions in a single ICO file. Add, remove, duplicate, and reorder frames with ease.

16x16 32x32 48x48 64x64 128x128 256x256

📁 Multiple Formats

Open ICO, PNG, BMP, WebP, JPG, and GIF files. Save as ICO with multi-frame support, or export as PNG and BMP.

Flip, rotate (90° or custom angle), resize frames, and insert external images. Full undo/redo history for every operation.

40+ color palette with recent colors tracking. Foreground/background dual color system with full alpha channel transparency support.

Built as a native Windows application with a sleek dark theme. Instant startup, zero configuration, fully portable.
