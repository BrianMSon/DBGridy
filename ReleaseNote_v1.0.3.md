DBGridy v1.0.3 Release Notes
============================

Date: 2025-12-16
Status: Maintenance Update

This release focuses on improving and optimizing internal DataGridView performance, as well as adding new icon resources.

---

## Improvements

### Supported Icons

Icon resources have been added for TreeView.

The following object icons are now supported:
- Table  
- View  
- Procedure  
- Trigger  

---

## Technical Changes

- Improved additional loading performance of DataGridView
